# 🐾 Clapet v1.0.0

**Clapet** — это милый и умный виртуальный питомец для рабочего стола. Он передвигается по экрану, реагирует на нажатия мышкой, озвучивает мысли нейросетью ElevenLabs, общается через ИИ-модели и повышает свой уровень от кормления печеньем.

Приложение полностью переписано на **Tauri 2.0 + React** для максимальной производительности, мгновенного запуска и сверхмалого потребления ОЗУ (~25 МБ вместо 200+ МБ в Electron).

## ✨ Что нового в версии 1.0.0

- 🚀 **Движок Tauri 2.0** — Мгновенный запуск и рекордно малое потребление ресурсов (~25 МБ ОЗУ).
- 🎙️ **Нейросетевая озвучка ElevenLabs (TTS)** — Профессиональный синтез речи для мыслей питомца и ответов ИИ.
- 🌐 **Полноценная локализация (EN / RU)** — Мгновенный переключатель между Английским и Русским языками для всего интерфейса и ответов.
- 🐾 **Выделенная вкладка настроек «Питомец»** — Настройка цвета, прозрачности (20%–100%), скорости дыхания, амплитуды и скорости ходьбы.
- 🎨 **Эластичные физические анимации** — Живое физическое дыхание, переминание лапками при ходьбе, покачивание головы при раздумьях, радостные прыжки и глубокий сон.
- 📈 **Система уровней и всплывающая полоска XP** — +10 XP за кормление печеньем. Прогрессивная система опыта (30, 60, 120, 240... XP) с плавно угасающей полоской над питомцем.
- 🔍 **Живой поиск в Лаунчере** — Мгновенная фильтрация параметров на всех вкладках настроек прямо во время ввода текста.
- 🛡️ **Защита от DevTools и Перезагрузок** — Полное отключение клавиш F12, F5, Ctrl+R, Ctrl+Shift+I/J/C и контекстного меню браузера.
- 🌐 **Встроенная поддержка Прокси** — Настройка HTTP, HTTPS и SOCKS5 прокси с автоматическим сохранением и проверкой пинга.
- 🤖 **Поддержка множества ИИ-провайдеров** — OpenAI, OpenRouter, DeepSeek, Groq, Ollama (локальные модели) и кастомные API эндпоинты.

## 🚀 Быстрый запуск (Для разработчиков)

### Требования

- [Node.js](https://nodejs.org/) (v18+)
- [Rust](https://www.rust-lang.org/) (stable toolchain)

### Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/S1sTeam/Clapet.git
cd Clapet/clapet-tauri

# Установить зависимости
npm install

# Запустить в режиме разработки (Vite + Tauri)
npm run tauri dev
```

### Сборка продакшен пакета

Чтобы скомпилировать бинарный файл для вашей текущей системы:

```bash
npm run tauri build
```

## 🎮 Как взаимодействовать с питомцем

- **Правая кнопка мыши (ПКМ)** по питомцу: Открывает круговое меню действий (Покормить, Спать/Разбудить, Гулять/Остановить, Спросить AI, Настройки ⚙️).
- **Зажатая ЛКМ**: Перетаскивание питомца в любую точку экрана.
- **Кормление**: Дать печеньку (кулдаун 15 минут). Каждая печенька даёт +10 XP и сопровождается эффектом частиц.
- **Спросить AI**: Введите запрос и нажмите **Enter** для получения короткого милого ответа с голосовой озвучкой.
